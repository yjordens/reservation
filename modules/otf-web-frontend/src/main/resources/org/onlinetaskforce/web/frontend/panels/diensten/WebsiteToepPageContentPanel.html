J2EE architecture

Onze expertise is gericht op de Java industry solutions. Waarom Java? Er zijn momenteel ontelbare toepassingen gebouwd met Java. Deze technologie heeft zich dus al goed bewezen. Daarnaast is Java ook een echte trendsetter (denk maar aan Google) en er is een onuitputtelijke open source markt die onnoemelijk veel bibliotheken gratis beschikbaar stelt met re-usable software. Dit houdt in dat we met Java solutions de kosten kunnen drukken.
Onze ontwikkelingen gebeuren ook altijd binnen een CI omgeving om naast een gegarandeerde oplevering ook een gegarandeerde kwaliteit te leveren.
